## ğŸ¯ PROJECT OVERVIEW

### What We're Building

A **visual node-based workflow builder** where users can:
- Drag and drop nodes to create AI agent workflows
- Connect nodes to create data pipelines
- Configure each node (API calls, AI processing, conditions)
- Deploy workflows to Solana blockchain
- Execute workflows automatically

### Why This Is Perfect for Rialo

| Rialo Feature | How We Showcase It |
|---------------|-------------------|
| Native Web Calls | Nodes fetch data from any API |
| Reactive Transactions | Workflows trigger automatically |
| Confidential Compute | API keys stored securely |
| AI Agent Economy | Multiple AI agents in pipelines |
| Automation | No manual execution needed |

### Tech Stack (All Free!)

```yaml
Frontend:
  Framework: React + Vite
  Canvas: React Flow
  Styling: Tailwind CSS
  Icons: Lucide React
  Wallet: Solana Wallet Adapter
  
Backend (Optional):
  Runtime: Node.js (for simulating execution)
  
Blockchain:
  Network: Solana Devnet (free testnet)
  Language: Rust (Anchor framework)
  Wallet: Phantom (free)
  
AI Models (Free):
  - Hugging Face Inference API (free tier)
  - DeepSeek via OpenRouter (free tier)
  - ChatGPT 3.5 alternatives (free)
```

**1. Solana CLI not found**

**2. Anchor build fails**
```bash
# Update Rust
rustup update

# Clean and rebuild
anchor clean
anchor build
```

**3. Wallet won't connect**
```javascript
// Make sure you have wallet adapter configured
import { WalletAdapterNetwork } from '@solana/wallet-adapter-base';
import { PhantomWalletAdapter } from '@solana/wallet-adapter-wallets';

const network = WalletAdapterNetwork.Devnet;
const wallets = [new PhantomWalletAdapter()];
```

**4. Transaction fails**
```javascript
// Check if you have SOL
const balance = await connection.getBalance(wallet.publicKey);
if (balance === 0) {
  // Get from faucet
  await connection.requestAirdrop(wallet.publicKey, 2 * LAMPORTS_PER_SOL);
}
```

**5. AI API rate limits**
```javascript
// Add retry logic with exponential backoff
async function callAIWithRetry(prompt, retries = 3) {
  for (let i = 0; i < retries; i++) {
    try {
      return await callAI(prompt);
    } catch (error) {
      if (i === retries - 1) throw error;
      await sleep(2 ** i * 1000); // 1s, 2s, 4s
    }
  }
}
```

**6. React Flow edges not showing**
```javascript
// Make sure you import the CSS
import 'reactflow/dist/style.css';

  default: CustomEdge,
```
